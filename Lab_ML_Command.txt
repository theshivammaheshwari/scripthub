# 🧪 ML Lab Setup - Run Instructions

This project uses a PowerShell script to set up the machine learning lab environment.

---

## ⚙️ Prerequisites

Make sure you have:

* Windows OS
* PowerShell installed (comes pre-installed on Windows)
* Administrator access (recommended)

---

## 📂 File Location

Ensure the setup script is placed on your Desktop:

```
Lab_ML_Setup.ps1
```

---

## ▶️ How to Run

Follow these steps:

1. Open **PowerShell**
2. Run the following command:

```
powershell -ExecutionPolicy Bypass -File "$env:USERPROFILE\Desktop\Lab_ML_Setup.ps1"
```

---

## ⚠️ Notes

* `ExecutionPolicy Bypass` allows the script to run even if script execution is restricted.
* If you face permission issues, try running PowerShell as **Administrator**.
* Make sure the file path is correct.

---

## ✅ Expected Outcome

* The script will automatically install and configure required ML tools and dependencies.
* After completion, your environment should be ready for use.

---

## ❓ Troubleshooting

* If script fails:

  * Check file name and location
  * Ensure PowerShell is updated
  * Run as Administrator
